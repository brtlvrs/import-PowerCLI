**import-PowerCLI**

A simple powershell script to load all VMware related modules and snapins.

### How do I get set up?
Download the repository to a folder. No setup needed.  
Open a PowerShell window and run the script.


#### Dependencies

	- PowerShell 3.0
	- PowerCLI > 5.x
